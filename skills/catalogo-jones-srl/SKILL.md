## 💬 Cómo usar este skill

### Para generar presupuesto (Agente F2)

Cuando un cliente pida presupuesto de service 20k para un MB 1634:

1. Calcular insumos: **$423.946** (del catálogo)
2. Calcular M.O.: 6-8 hs × tarifa del taller = **$600.000**
3. Sumar subtotal: **$1.023.946** s/IVA
4. Agregar IVA 21%: **$215.029**
5. Total final: **$1.238.975** c/IVA

Devolver presupuesto formal con todos los conceptos discriminados.

### Para reposición de stock (Agente 3)

Cuando el nivel de un insumo esté por debajo del mínimo:

1. Identificar el item
2. Buscar precio actualizado en el catálogo
3. Generar orden de compra con cantidad sugerida
4. Enviar por WhatsApp a Jones con: referencia del item, cantidad, urgencia

### Para recepción de unidades (Agente F1)

Cuando ingresa un camión por service, pre-calcular el costo esperado
usando el catálogo así Bruno sabe en qué está comprometido antes de
empezar el trabajo.
